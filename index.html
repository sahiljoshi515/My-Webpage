Featured Projects

These are some of the projects I have worked on recently.

ReachOUT

Our frontend scrum team worked to design a better UI using React.js for web app and React Native for mobile app. ReachOUT is an application which will be used by students to schedule appointments with advisors.

Whack-A-MoLED

I worked with a friend to simulate the famous arcade game - whack-a-mole by leveraging arduino and electronics. We simulate the hammer by using accelerometer and moles by using LEDs. You can find more information here.

Simon Game

It is a memory game which is designed with HTML and CSS. I used Javascript and JQuery to achieve the functionality. I built this game over Summer 2020 to gain experience in building web applications using basic web languages.


To-Do-List Web App

You can create different categories of work/days by adding the name as the new endpoint to the url. I used EJS for the frontend and Node.js along with MongoDB for the backend. I have deployed it on Heroku.

Guess The Flower

I used the CoreML framework to build this application in XCode. It takes an image of a flower as the input and gives the name as the output based on the Inception v3 Model designed by Google. I wrote this program in Swift.

Bitcoin Price Reporter

This app will give you the latest Bitcoin prices in all the popular, traded currencies, so you can track how well your Bitcoin investments are doing on the go. I used coinapi.io to fetch data using JSONDecoder for JSON parsing.
